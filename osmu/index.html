- 리서치 요약본에 없는 사실을 추가하지 않는다.
- 제목 5개를 제안한다.

4교시 PASS 기준

Prompt · 숏폼 대본/컷 구성

30-60초 분량으로 구성하고, 각 컷마다 다음을 작성해줘.

- 내레이션
- 자막 문장

규칙:
- 전체를 설명하려 하지 말고 한 가지 메시지만 잡는다.
- 리서치 요약본에 없는 사실을 추가하지 않는다.
- 과장 표현은 피한다.
Command · Pixverse image-to-video
REAL=/opt/homebrew/lib/node_modules/pixverse/dist/index.js

node "$REAL" create video \
  --image "cut-01.png" \
  --prompt "Subtle cinematic motion, slow push-in, gentle parallax, keep composition stable, no text, no logos, no new objects. Maintain the original visual metaphor." \
  -m v6 \
  -d 5 \
  --aspect-ratio 9:16 \
  --no-audio \
  --json
생성 지연 대응
Pixverse 영상 생성이 수업 중 늦어지면 “30-60초 숏폼 대본 + 컷별 프롬프트 + 실행 명령 또는 job 기록”을 기본 PASS로 인정합니다. 최소 1개 컷 영상이나 최종 영상 파일/URL이 있으면 완성 PASS입니다.

자주 막히는 지점

  • 리서치 질문이 너무 넓다.
  • 요약본 없이 블로그를 바로 쓴다.
  • 카드뉴스 한 장에 메시지를 여러 개 넣는다.
  • 숏폼에 전체 내용을 다 넣으려 한다.
  • Pixverse 생성이 늦어져 수업 흐름이 끊긴다.

대응 원칙

  • 분야·타깃·채널을 좁힌다.
  • 요약본으로 돌아가 사실/해석을 분리한다.
  • 카드당 한 메시지로 자른다.
  • 숏폼은 한 각도만 선택한다.
  • 생성 job 기록을 남기고 수업 후 완성한다.
