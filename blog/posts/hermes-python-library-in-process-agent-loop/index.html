Hermes Python LibraryлҠ” м—җмқҙм „нҠё лЈЁн”„лҘј м•ұ м•ҲмңјлЎң л„ЈлҠ”лӢӨ

Hermes Agent can run in-process as a Python library through AIAgent, without a CLI or subprocess. This guide covers chat(), run_conversation(), tool boundaries, state handling, and concurrency safety.

hermesai-agentworkflowpython-library

Hermes AgentлҘј м“°кё° мң„н•ҙ лҠҳ н„°лҜёл„җмқҙлӮҳ л©”мӢ м ҖлҘј м—ҙ н•„мҡ”лҠ” м—ҶлӢӨ. Python н”„лЎңк·ёлһЁм—җм„ң AIAgentлҘј м§Ғм ‘ importн•ҳл©ҙ лӘЁлҚё нҳём¶ң, tool call, мһ¬мӢңлҸ„к№Ңм§Җ нҸ¬н•Ён•ң м—җмқҙм „нҠё лЈЁн”„лҘј к°ҷмқҖ н”„лЎңм„ёмҠӨ м•Ҳм—җм„ң мӢӨн–үн•  мҲҳ мһҲлӢӨ. кё°мЎҙ м•ұмқҙ мҡ”мІӯ мҲҳлӘ…кіј м¶ңл Ҙ нҳ•мӢқмқ„ л§Ўкі , Hermesк°Җ нҢҗлӢЁкіј лҸ„кө¬ мӮ¬мҡ©мқ„ л§Ўм•„м•ј н•  л•Ң м“°лҠ” л°©мӢқмқҙлӢӨ.

л¬ҙм—Үмқҙ л“Өм–ҙ мһҲлӮҳ

к°ҖмһҘ лӢЁмҲңн•ң мһ…кө¬лҠ” chat()мқҙлӢӨ. л¬ёмһҗм—ҙ н•ҳлӮҳлҘј л„ҳкё°л©ҙ Hermesк°Җ н•„мҡ”н•ң лҸ„кө¬лҘј нҳём¶ңн•ң л’Ө мөңмў… лӢөліҖ л¬ёмһҗм—ҙл§Ң л°ҳнҷҳн•ңлӢӨ. мӢӨн–ү кіјм •кіј л©”мӢңм§Җ мқҙл Ҙк№Ңм§Җ лӢӨлӨ„м•ј н•ңлӢӨл©ҙ run_conversation()мқ„ м“ҙлӢӨ. л°ҳнҷҳк°’м—җлҠ” final_responseмҷҖ system, user, assistant, tool callмқ„ нҸ¬н•Ён•ң messagesк°Җ л“Өм–ҙк°„лӢӨ.

мқҙ л°©мӢқмқҖ мҷёл¶Җ н”„лЎңнҶ мҪңкіј кө¬л¶„н•ҙм•ј н•ңлӢӨ. ACPлҠ” IDEк°Җ HermesлҘј м ңм–ҙн•  л•Ң, API ServerлҠ” лӢӨлҘё м–ём–ҙлӮҳ OpenAI нҳёнҷҳ нҒҙлқјмқҙм–ёнҠёк°Җ HTTPлЎң м ‘к·јн•  л•Ң л§һлӢӨ. Python LibraryлҠ” subprocessлӮҳ л„ӨнҠёмӣҢнҒ¬ кІҪкі„ м—Ҷмқҙ м•ұкіј к°ҷмқҖ л©”лӘЁлҰ¬ кіөк°„м—җм„ң AIAgentлҘј мӢӨн–үн•ңлӢӨ. к·ёлһҳм„ң л¶ҷмқҙкё°лҠ” мүҪм§Җл§Ң мғҒнғңмҷҖ лҸҷмӢңм„ұлҸ„ нҳёмҠӨнҠё м•ұмқҙ мұ…мһ„м ём•ј н•ңлӢӨ.

м•ұ м•Ҳм—җм„ң мӢӨн–үн•ҳлҠ” лІ•

кіөмӢқ м ҖмһҘмҶҢлҘј нҳ„мһ¬ Python нҷҳкІҪм—җ м„Өм№ҳн•ңлӢӨ.

CLIм—җм„ң м“°лҚҳ provider мқёмҰқкіј нҷҳкІҪ ліҖмҲҳлҠ” лқјмқҙлёҢлҹ¬лҰ¬м—җм„ңлҸ„ к·ёлҢҖлЎң н•„мҡ”н•ҳлӢӨ. нӮӨлҘј мҪ”л“ңм—җ м Ғм§Җ л§җкі  Hermes м„Өм •мқҙлӮҳ н”„лЎңм„ёмҠӨ нҷҳкІҪм—җм„ң мқҪкІҢ н•ңлӢӨ. мөңмҶҢ мӢӨн–ү мҪ”л“ңлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

modelмқ„ мғқлһөн•ҳл©ҙ Hermes м„Өм •мқҳ кё°ліё лӘЁлҚёмқ„ мӮ¬мҡ©н•ңлӢӨ. м•ұм—җм„ң лӘЁлҚёкіј providerлҘј кі м •н•ҙм•ј н•ҙлҸ„ мһҗкІ© мҰқлӘ… к°’к№Ңм§Җ мғқм„ұмһҗм—җ л°•м•„ л„Јм§ҖлҠ” м•ҠлҠ”лӢӨ. лӘЁлҚё м„ нғқмқҖ мҪ”л“ңм—җ л‘ҳ мҲҳ мһҲм§Җл§Ң, л№„л°Җ к°’мқҖ м„Өм •мқҙлӮҳ нҷҳкІҪ ліҖмҲҳм—җ лӮЁкІЁ л‘җлҠ” нҺёмқҙ көҗмІҙмҷҖ нҡҢмҲҳк°Җ мүҪлӢӨ.

from run_agent import AIAgent

agent = AIAgent(
    quiet_mode=True,
    enabled_toolsets=["web"],
    skip_context_files=True,
    skip_memory=True,
    max_iterations=12,
)

answer = agent.chat("Python 3.13мқҳ мЈјмҡ” ліҖкІҪм җмқ„ м°ҫм•„ мҡ”м•Ҫн•ҙ мӨҳ")
print(answer)

quiet_mode=TrueлҠ” spinnerмҷҖ tool progress к°ҷмқҖ CLI м¶ңл Ҙмқ„ м–өм ңн•ңлӢӨ. enabled_toolsetsлҠ” н—Ҳмҡ© лӘ©лЎқмқҙкі , disabled_toolsetsлҠ” лҢҖл¶Җл¶„мқҳ лҸ„кө¬лҘј м—ҙм–ҙ л‘җлҗҳ мқјл¶Җл§Ң л§үмқ„ л•Ң м“ҙлӢӨ. мҷёл¶Җ мҡ”мІӯмқ„ мІҳлҰ¬н•ҳлҠ” stateless endpointлқјл©ҙ skip_context_files=TrueмҷҖ skip_memory=TrueлЎң мһ‘м—… нҸҙлҚ”мқҳ AGENTS.mdмҷҖ мһҘкё° кё°м–өмқҙ мҡ”мІӯм—җ м„һмқҙм§Җ м•ҠкІҢ н•  мҲҳ мһҲлӢӨ.

м—¬лҹ¬ н„ҙмқ„ мқҙм–ҙм•ј н•ңлӢӨл©ҙ мқҙм „ кІ°кіјмқҳ л©”мӢңм§Җ лӘ©лЎқмқ„ лӢӨмӢң л„ҳкёҙлӢӨ.

first = agent.run_conversation("кІҖнҶ  кё°мӨҖмқ„ лӢӨм„Ҝ мӨ„лЎң м •лҰ¬н•ҙ мӨҳ")
second = agent.run_conversation(
    "к·ё кё°мӨҖмқ„ JSON н•ӯлӘ©мңјлЎң л°”кҝ” мӨҳ",
    conversation_history=first["messages"],
)
print(second["final_response"])

run_conversation()мқҖ м „лӢ¬л°ӣмқҖ historyлҘј лӮҙл¶Җм—җм„ң ліөмӮ¬н•ҳлҜҖлЎң мӣҗліё лӘ©лЎқмқ„ м§Ғм ‘ л°”кҫём§Җ м•ҠлҠ”лӢӨ. н•ңнҺё ephemeral_system_promptлҠ” м•ұ м „мҡ© м—ӯн• мқ„ мӨ„ л•Ң м“ё мҲҳ мһҲкі , save_trajectories=TrueлҠ” мӢӨн–ү кё°лЎқмқ„ ShareGPT нҳ•мӢқ JSONLлЎң лӮЁкёҙлӢӨ. кё°лЎқм—җлҠ” лҜјк°җн•ң мһ…л Ҙмқҙ л“Өм–ҙк°Ҳ мҲҳ мһҲмңјлӢҲ кё°ліёк°’мІҳлҹј кәј л‘” мғҒнғңм—җм„ң н•„мҡ”м„ұмқ„ лЁјм Җ нҢҗлӢЁн•ҳлҠ” нҺёмқҙ лӮ«лӢӨ.

мӢӨм ң мӮ¬мҡ© мӮ¬мқҙл“ңл°”

мқҙ мӮ¬мҡ©мһҗмқҳ м…Ӣм—…м—җм„ңлҠ” м§ҖмӢқ м •лҰ¬, к°ңл°ң кІҖмҰқ, мҡҙмҳҒ нҷ•мқёмқ„ м„ңлЎң лӢӨлҘё profileкіј workerлЎң лӮҳлҲ„кі , мҳҲм•Ҫ мһ‘м—…мқҖ лі„лҸ„ session sourceмҷҖ мҷ„лЈҢ кё°лЎқмқ„ лӮЁкёҙлӢӨ. к°Ғ м—ӯн• мқҙ к°ҷмқҖ лҢҖнҷ” мқҙл Ҙкіј tool surfaceлҘј л¬ҙмЎ°кұҙ кіөмң н•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ, кІ°кіјк°Җ м–ҙлҠҗ мӢӨн–ү кІҪкі„м—җм„ң лӮҳмҷ”лҠ”м§ҖлҸ„ л’Өм—җ нҷ•мқён•  мҲҳ мһҲлӢӨ. лё”лЎңк·ё л°ңн–үмқҙлӮҳ кёҙ л№Ңл“ңмІҳлҹј мӢңк°„мқҙ кұёлҰ¬лҠ” мқјмқҖ м¶”м Ғ к°ҖлҠҘн•ң background нқҗлҰ„м—җ л‘җкі , мӮ¬лһҢмқҙ лҢҖнҷ”н•ҳлҠ” м°ҪмқҖ мғҲ м§ҖмӢңлҘј л°ӣмқ„ мҲҳ мһҲкІҢ л№„мӣҢ л‘”лӢӨ. мқҙлҹ° нҷҳкІҪм—җм„ң Python embeddingмқҖ м „мІҙ мҡҙмҳҒмқ„ лҢҖмӢ н•ҳлҠ” мғҒмЈј agentліҙлӢӨ м§§мқҖ кІҖмҰқкё°мҷҖ лӮҙл¶Җ endpointм—җ м–ҙмҡёлҰ¬лҜҖлЎң, мҡ”мІӯл§ҲлӢӨ мғҲ AIAgentлҘј л§Ңл“Өкі  н•„мҡ”н•ң toolsetл§Ң м—ҙл©° мҳӨлһҳ мқҙм–ҙм§Ҳ мқјмқҖ GatewayлӮҳ Kanbanм—җ лӮЁкё°лҠ” мӢқмқҙлӢӨ.

н•Ём •кіј нҢҒ

  • н•ҳлӮҳмқҳ AIAgentлҘј м—¬лҹ¬ threadк°Җ кіөмң н•ҳл©ҙ м•Ҳ лҗңлӢӨ. лҢҖнҷ” мқҙл Ҙ, tool session, iteration counterк°Җ instance м•Ҳм—җ мһҲмңјлҜҖлЎң threadлӮҳ taskл§ҲлӢӨ мғҲ instanceлҘј л§Ңл“ лӢӨ.
  • enabled_toolsetsлҘј лЁјм Җ кі л Өн•ңлӢӨ. нҢҢмқјл§Ң мқҪлҠ” кІҖмӮ¬кё°м—җ terminalкіј browserк№Ңм§Җ м—ҙ мқҙмң лҠ” м—ҶлӢӨ.
  • лӢЁмҲң endpointм—җм„ңлҠ” max_iterationsлҘј лӮ®м¶° л¬ҙн•ңн•ң tool loopмҷҖ мҳҲмғҒ л°–мқҳ л№„мҡ©мқ„ м ңн•ңн•ңлӢӨ.
  • skip_context_filesмҷҖ skip_memoryлҠ” ліҙм•Ҳ мҠӨмң„м№ҳмқҙл©ҙм„ң кё°лҠҘ мҶҗмӢӨмқҙкё°лҸ„ н•ҳлӢӨ. н”„лЎңм қнҠё к·ңм№ҷмқҙлӮҳ мӮ¬мҡ©мһҗ м„ нҳёк°Җ н•„мҡ”н•ң мһ‘м—…м—җм„ңлҠ” л¬ҙмЎ°кұҙ лҒ„м§Җ м•ҠлҠ”лӢӨ.
  • кёҙ н”„лЎңм„ёмҠӨм—җм„ңлҠ” к°Ғ conversationмқҙ м •мғҒ мў…лЈҢлҗҳлҸ„лЎқ мҳҲмҷёмҷҖ timeoutмқ„ лӢӨлЈ¬лӢӨ. к·ёлһҳм•ј terminal sessionкіј browser к°ҷмқҖ лҰ¬мҶҢмҠӨ м •лҰ¬лҸ„ лҒқлӮңлӢӨ.

м•ұмқҙ PythonмңјлЎң мһ‘м„ұлҗҳм–ҙ мһҲкі  мҡ”мІӯл§ҲлӢӨ лҸ…лҰҪлҗң м—җмқҙм „нҠё мӢӨн–үмқҙ н•„мҡ”н•ҳлӢӨл©ҙ Library л°©мӢқмқҙ к°ҖмһҘ м§§лӢӨ. м—¬лҹ¬ м–ём–ҙм—җм„ң нҳём¶ңн•ҳкұ°лӮҳ н”„лЎңм„ёмҠӨлҘј л¶„лҰ¬н•ҙм•ј н•ҳл©ҙ API Serverк°Җ лӮ«кі , IDE м—°кІ°м—җлҠ” ACPк°Җ л§һлӢӨ. к°ҷмқҖ н”„лЎ¬н”„нҠёлҘј лҢҖлҹүмңјлЎң мІҳлҰ¬н•ҳлҠ” мқјмқҙлқјл©ҙ м§Ғм ‘ threadлҘј мЎ°лҰҪн•ҳкё°ліҙлӢӨ Hermesмқҳ batch runnerлҘј лЁјм Җ кІҖнҶ н•ҳлҠ” нҺёмқҙ м•Ҳм „н•ҳлӢӨ.
