MICE / Outdoor Event Safety

MICE 행사 안전 적용성 체크리스트

행사 조건을 입력하면 법령·조례 후보와 위험요인을 정리하고, Nemotron-Personas-Korea 기반 합성 관람객 코호트로 계획의 사람 중심 사각지대를 함께 점검합니다.

네트워크 법령 조회 없음

이 페이지는 함께 배포된 JSON 온톨로지만 읽어 브라우저 안에서 계산합니다. 법률 자문이나 관할기관 승인을 대체하지 않습니다.

행사 조건 입력

300명 이상이면 동선·의료 사전검토가 권고되고, 1,000명 이상이면 의무가 더 늘어납니다.

실제 참석자 예측이 아니라 합성 사용자 관점의 계획서 QA입니다.

전문가 입력 (행사 유형·베뉴·관할 직접 지정)
온톨로지 데이터를 불러오는 중입니다.

초기 로드 후 샘플 체크리스트가 자동 생성됩니다.
